# Local Fonts Directory

Please download the following font files from Google Fonts helper or google webfonts helper (https://gwfh.dev/) and save them here as `.woff2`:

## Inter Font Family:
* `Inter-Light.woff2` (weight 300)
* `Inter-Regular.woff2` (weight 400)
* `Inter-Medium.woff2` (weight 500)
* `Inter-SemiBold.woff2` (weight 600)
* `Inter-Bold.woff2` (weight 700)

## Cormorant Garamond Font Family:
* `CormorantGaramond-Regular.woff2` (weight 400)
* `CormorantGaramond-Italic.woff2` (weight 400 italic)
* `CormorantGaramond-Medium.woff2` (weight 500)
* `CormorantGaramond-MediumItalic.woff2` (weight 500 italic)
* `CormorantGaramond-SemiBold.woff2` (weight 600)
* `CormorantGaramond-SemiBoldItalic.woff2` (weight 600 italic)
* `CormorantGaramond-Bold.woff2` (weight 700)
